Flutter Based Quiz App

A web app for creating and playing quizzes

Link to Github Repo | Link to App

What I Built

A full-stack Jeopardy-style Flutter web app where my friends and I create and play custom quizzes together. Think of it as our own private quiz show - we take turns being quizmaster and contestants, complete with buzzer functionality and score tracking.
My friend group loves trivia nights, but existing apps never quite fit what we wanted. So instead of settling for generic quiz platforms, I decided to learn Flutter and build exactly what we need. Classic developer move: "I can build that better myself."

How It Works

Custom Quiz Creation: Build question sets right in the app with a drag-and-drop builder, or bulk-import them from a CSV file. Questions can carry images, audio clips, or embedded YouTube videos - not just text.
Community Marketplace: Browse public quizzes other people have created, search and filter by name or creator, and add any set to your own collection to host with your friends.
Live Game Experience: One person signs in and hosts as quizmaster, creating a game room with a unique code. Others join with the code (or jump in as a guest, no account needed), then race to answer off the board - Jeopardy! style gameplay.
Buzzer Mechanic: Players buzz in and the app timestamps every press, so the fastest finger genuinely wins.
Real-time Scoring: A live leaderboard tracks everyone's performance throughout the game, with custom scoring because the quizmaster is god. Chain multiple boards into one session and cap it off with a final standings screen and a celebratory confetti finish.
Web-based: Runs in any browser, so no app downloads needed - just share the link and everyone's ready to play.

Technical Journey

First time diving deep into Flutter, so this was a crash course in web app development. I built the entire frontend, handled state management, and set up Firebase as the backend - Auth for secure sign-in (plus guest mode), Firestore and Realtime Database for live game state, Storage for question media, and App Check with reCAPTCHA to keep things safe. The buzzer was particularly fun to get right: every press is timestamp-ordered so the fastest responder is picked fairly across multiple players. On top of the core game loop I built an in-app quiz builder with drag-and-drop question tabs, CSV import, a community marketplace for publishing and downloading sets, and post-game profile stats with score graphs (fl_chart). PostHog analytics rounds it out so I can see how the app actually gets used.

Impact and Future Plans

The real win isn't the code - it's the quality time with friends. We've had some epic quiz nights thanks to this app, and everyone's gotten surprisingly competitive about creating better questions. Now that there's a proper backend, a live buzzer, and a community marketplace, I'm excited to turn this into a standalone product for quiz enthusiasts everywhere. Next up: Excel (.xlsx) import alongside CSV, deeper player profiles and analytics, real-time chat during games, more question formats, and eventually native mobile apps.
